Global setup:
 Set up git
  git config --global user.name "Your Name"
  git config --global user.email ggvjava@yahoo.com
      
Next steps:
  mkdir mom
  cd mom
  git init
  touch README
  git add README
  git commit -m 'first commit'
  git remote add origin git@github.com:ggvjava/mom.git
  git push -u origin master
      
Existing Git Repo?

  git remote add origin git@github.com:ggvjava/mom.git
  git push -u origin master
